# 个人作品集网站开发指令

## 项目概述
使用 Vue3 (Composition API) + TypeScript + Tailwind CSS 开发的个人作品集网站，基于 Vite 构建工具，遵循 Vue3 官方推荐的开发范式。

## 开发规范
- 使用 Vue3 组合式 API 开发组件;优先使用 Vue 内置的响应式 API
- 使用 Tailwind CSS 编写样式
- 组件要可复用
- 代码要有注释

## 设计要求
- 深色主题（背景 #0a0a0a，文字 #ffffff）
- 使用渐变色作为强调色
- 添加平滑的滚动动画
- 确保移动端适配

## 注意事项
- 保持设计简洁，不要过度设计
- 性能优化：图片使用懒加载
- 确保所有链接可点击
