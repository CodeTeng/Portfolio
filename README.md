# Personal Portfolio Website

A modern, responsive personal portfolio website built with **Vue 3**, **TypeScript**, **Vite**, and **Tailwind CSS**.

## âœ¨ Features

- **Modern Tech Stack**: Built with Vue 3 (Composition API) and TypeScript.
- **Styling**: Utility-first styling with Tailwind CSS v4.
- **Responsive Design**: Fully responsive layout for mobile, tablet, and desktop.
- **Dark/Light Mode**: Built-in theme toggler with persistent state.
- **Animations**: Smooth scroll animations using `@vueuse/motion`.
- **Interactive Components**:
  - **Hero Section**: Dynamic text and avatar with entrance animations.
  - **About Section**: Skill progress bars and personal statistics.
  - **Projects Section**: Paginated project showcase with hover effects.
  - **Footer**: Social links and copyright info.

### Prerequisites

- Node.js (v16.0 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
